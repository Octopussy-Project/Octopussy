#
# Software Requirements
#

Before installing Octopussy, be sure to have installed: 
- Apache2
- Apache::ASP
- Perl
- Mod_Perl
- Mod_SSL
- Mysql
- Nscd
- RRDTool & librrd0
- Rsyslog
- Sudo
- htmldoc

and these Perl modules (from CPAN):
- App::Info
- Cache::Cache
- Crypt::PasswdMD5
- Date::Manip
- DateTime::Format::Strptime
- DBD
- DBD::mysql
- DBI
- File::Slurp
- JSON
- Linux::Inotify2
- List::MoreUtils
- Locale::Maketext::Lexicon
- Locale::Maketext::Simple
- LWP
- Mail::Sender
- Net::FTP
- Net::LDAP
- Net::SCP
- Net::Telnet
- Net::XMPP
- Path::Tiny
- Proc::PID::File
- Proc::ProcessTable
- Readonly
- Regexp::Assemble
- Sys::CPU
- Term::ProgressBar
- Unix::Syslog
- URI
- version
- XML::Simple (Take a look http://search.cpan.org/~grantm/XML-Simple-2.18/lib/XML/Simple.pm#ENVIRONMENT for possible performance issues)

Good Luck !

For more information about Octopussy installation:
https://github.com/sebthebert/Octopussy_Documentation/blob/master/01_Installation.md
